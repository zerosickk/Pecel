Pecel Lele Management System
Aplikasi web management system yang dirancang khusus untuk warung pecel lele. Sistem ini menyediakan dashboard untuk monitoring pesanan, pendapatan, dan statistik bisnis secara real-time. Dibangun dengan teknologi modern dan mengutamakan keamanan serta kemudahan penggunaan. Interface yang responsive memungkinkan akses dari berbagai perangkat, baik desktop maupun mobile.

Installation

Clone repository

git clone https://github.com/zerosickk/Pecel.git
cd Pecel

# Install dependencies
npm install

# Start application
npm start
Open browser: http://localhost:3000
